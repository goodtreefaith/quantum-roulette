# Code of Conduct

Quantum Roulette welcomes contributors who want to improve the project for community raffles, classrooms, streams, events, and browser-game experiments.

## Expected Behavior

- Be respectful and specific in issues, pull requests, and reviews.
- Assume good intent while still asking for evidence and clear reproduction steps.
- Keep feedback focused on the code, docs, design, accessibility, security, or user impact.
- Make room for new contributors by explaining decisions and linking context.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language
- Personal attacks or repeated bad-faith arguments
- Publishing private information without consent
- Security reports posted publicly before the maintainer has had time to review them

## Reporting

For conduct issues, contact the maintainer through the GitHub profile listed for this repository. For sensitive security reports, follow `SECURITY.md` instead of opening a public issue.

The maintainer may remove comments, close threads, block accounts, or decline contributions that make the project unsafe or unwelcoming.

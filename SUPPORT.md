# Support

Use GitHub issues for public project support.

## Bugs

Open a bug report with:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser and device details
- Screenshots, recordings, or console output when useful

## Feature Requests

Open a feature request and describe the event, classroom, stream, or community workflow the change would support.

## Security

Do not open public issues for sensitive security reports. Follow `SECURITY.md`.

## Maintainer Scope

Quantum Roulette is maintained as an open-source browser project. Support is best-effort and focused on reproducible bugs, security issues, accessibility, documentation, and release readiness.
